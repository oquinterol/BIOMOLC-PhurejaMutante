## Instalación

    ```bash
    make trim
    ```
## Uso
El uso se explica a mas detalle en la documentación pero al hacer uso de **make** se puede acceder a todas las opciones.

- Limpieza y Filtrado

    ```bash
    make trim
    ```
- DEA: Análisis de Expresión Diferencial

    ```bash
    make dea
    ```
- SNPs: Llamado de variantes

    ```bash
    make snp
    ```
